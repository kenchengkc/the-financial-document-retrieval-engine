# Data Directory

`data/sample/` is for tiny committed fixtures and the generated NASDAQ/NYSE company seed list in `listed_companies.json` (one CIK per company, ETFs excluded).

`data/raw/`, `data/cache/`, and `data/processed/` are ignored except for `.gitkeep` files. Use them for downloaded SEC filings, HTTP cache files, parsed artifacts, embeddings, and generated indexes.
